# yusaku-app-server

Yusaku-Appのサーバサイド実装です。


## ローカル開発環境

1. `git clone --recursive https://github.com/<Github id>/yusaku-app-server.git`
2. `cd yusaku-app-server`
3. `npm install && cd yusaku-app && npm install && cd ..`
4. `npm run dev`

## Heroku production環境
1. `heroku create`
2. `git push heroku master`
